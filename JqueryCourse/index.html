What is Jquery?

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript. Jquery Definition

Jquery Syntax

Dollor Sign $

in jquery all the commands start with $(dollor sign) and end with ;(semicolon) for Example:

$('element').hide( );

Jquery Selectors

in jquery to do a action on a element in our document we have first select that element, absolutly like js.

select a Element and do some action on that...

in jquery selector is like css selector. class selector, id selector etc.

Tag Name Selector

in tag name selector we have give a html tag name and in return jquery give us a array!

$('div');
NOTE: when we use tag name method we get a array, because we may have several elements with that name.

Class Name Selector

in class name selector we have give a class and like tag name selector in return Jquery give us a array!

as i told you in Jquery selectors is like css selector.so we put a dot before class name.

$('.class');
NOTE: when we use class name method we get a array, because we may have several elements with that name.

Id Selector

in id selector, as always we give a value. in this case we have give a id.

in id selector again like class name selector we have put a #(sharp) before our id to tell jquery our mean is an id.

$('#id');
NOTE: when we use id selector method we get an single element, because we should have one element with that id.

Other Selector

as i told you in jquery, selectors is like css selector. so we can select an element in diffrent ways.

for Example:

$('div p'); // all p elements in div elements
$('div.division'); // all div elements with division class name

Jquery Html Manipulation

jquery have some tools for manipulating DOM.

html() Method

this method work same innerHtml property in JS.

with this method you cant get or set value on a target. i will try both

geting value in a tag :

<div id='division1'>
<p> Hello World </p>
</div>
<script>
$('#division1').html(); // return <p> Hello World </p>
</script>

seting value of a tag :

<div id='division1'>
<p> Hello World </p>
</div>
<script>
$('#division1').html('test'); // now the #division1 content is ==> 'test'
</script>

text() Method

this method work same innerText property in JS.

with this method you cant get or set value on a target. i will try both

geting value in a tag :

<div id='division1'>
<p> Hello World </p>
</div>
<script>
$('#division1').value(); // return <p> Hello World </p>
</script>

seting value of a tag :

<div id='division1'>
<p> Hello World </p>
</div>
<script>
$('#division1').value('test'); // now the #division1 content is ==> 'test'
</script>
NOTE: what is diffrence between html() and text() ???
html() method get the content with html tag and you can set html tag with this method
but text() method just get content without html tags and you can just set content, no html tag!
exactly same diffrence between innerHtml and innerText in JS

Jquery Add Element To DOM

in jquery adding element to DOM tree is so easy!

jquery have several method to do this, but before adding a element in dom you have create that element then add it with help of jquery methods

append() method

this method append a element or content for us in a specefic element

in this case we create e element and then add into DOM tree.

let element = $('<p> </p>').text('Hello World');
$('#division1').append(element);

in this example i first create a element and then save that in to a element with 'element' name
then with append Jquery method append that to #division1 element

prepend() method

this method append a element or content for us at the start of the specefic element

by default added element go to the end of the parent tag, for this reason we just have prepend method.

in this case we create e element and then add into DOM tree.

let element = $('<p> </p>').text('Hello World');
$('#division1').prepend(element);

in this example i first create a element and then save that in to a element with 'element' name
then with prepend Jquery method append that at the start of the #division1 element

after() method

this method append a element or content for us after specefic element.

in this case we create e element and then add into DOM tree.

let element = $('<p> </p>').text('Hello World');
$('#division1').after(element);

in this example i first create a element and then save that in to a element with 'element' name
then with after Jquery method append that after #division1 element

before() method

this method append a element or content for us before specefic element.

in this case we create e element and then add into DOM tree.

let element = $('<p> </p>').text('Hello World');
$('#division1').before(element);

in this example i first create a element and then save that in to a element with 'element' name
then with after Jquery method append that before #division1 element

Jquery Animations

jquery also have some animations that you can use that.

hide() and show() Methods

with these methods you can hiding and showing a element.

actually these methods changing display property of the target element.

$('#division1').hide(); // hiding #division1 element $('#division1').show(); // showing #division1 element

fadeIn(), fadeOut(), fadeToggle() Method

with these Methods you can fading an element.

fadeIn method show an element with fade effect, fadeOut method hiding an element with fade effect, fadeToggle do toggling between hide and show status

$('#division1').fadeIn(); // fade In #division1 element ===> showing element
$('#division1').fadeOut(); // fade Out #division1 element ===> hiding element
$('#btn').click(() => {
$('#division1').fadeToggle(); // toggling between hide and show status
});

slideDown(), slideUp(), slideToggle() Method

with these Methods you can sliding an element.

slideDown method show an element with slide effect, slideUp method hiding an element with slide effect, slideToggle do toggling between hide and show status

$('#division1').slideDown(); // show #division1 element
$('#division1').fadeOut(); // hide #division1 element
$('#btn').click(() => {
$('#division1').slideToggle(); // toggling between hide and show status
});
NOTE: all these methods accept two argument. first is a callback function that execute after action finished, second is an milisencond that is animation time.

Jquery Css Animation

with jquery seting animation is so easy.

animation() Method

jquery have a method called animation that do all actions.

this method accept three argument. first is animation css property for making a move, second is animation duration time, third is callback function that execute after animations finished.

$('#division1').animation({
top: '20px', left: '20px'
}, 2000, () => {
alert('animation is over!');
});

in this example #division1 first move in 2second, and at the end throw an alert().

Stoping Animation Before End

also you can stop your animation before end that with stop() method.

$('#btn').click(() => {
$('#division1').stop();
});

in this example whene clicked on #btn, #division1 animation has been stop

Jquery Finding and Element

for finding elements in DOM tree, you can use jquery methods.

parent() Method

with this method you can get parent element.

$('#division1').parent();

parents() Method

with this method you can get all parent elements.

this method also accept a argument.
you can pass a selector for example

$('#division1').parents(); // return all parents $('#division1').parents('.parent'); // return all parents with .parent class name

also you can find childerens of element.

children() Method

with this method you can find childrens of a element. this method accept an argument.

$('#division1').children(); // return all childrens $('#division1').children('.children'); // return all children with .children class name

you can find sideway element with jquery methods.

siblings() Method

this method return all sideway elements.

$('#division1').siblings(); // return all sideway elements

next() Method

this method return first next element.

$('#division1').next(); // return first sideway elements

nextAll() Method

this method return all next elements.

$('#division1').nextAll(); // return all sideway elements

nextUntill() Method

this method return all next elements between two element.

$('#division1').nextUntill('p'); // return all sideway elements between #division1 and div tag

prev() Method

this method return first previous elements.

$('#division1').prev('p'); // return first previous elements.

prevAll() Method

this method return all previous elements.

$('#division1').prevAll('p'); // return all previous elements.

prevUntill() Method

this method return all prev elements between two element.

$('#division1').prevUntill('p'); // return all previous elements between #division1 and div tag
